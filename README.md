## Консольный ToDo-лист на Go

Простой, быстрый и вечный ToDo-лист прямо в терминале.
Задачи сохроняются в 'tasks.json' - и не пропадают после закрытия!

## Особенности
- Добовление задач
- Отметка выполненных задач галочкой✅
- Автоматическое сохранение и загрузка задач из JSON
- Чистая архитектура: 'cmd' + 'internal/todo'

## Технологии 
- Go 1.25.4
- JSON для хранения и загрузки данных
- Стандартные библиотеки:
    encoding/json;
    bufio;
    os
